# nodejs-web-elm-embedded-make-reactor-hello-world

## Description
A POC elm program hello world.
Uses `make` and `reactor` commands to
compile elm files to html.

## Tech stack
- npm
    - elm

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
- [Availble](http://localhost/index.html)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
